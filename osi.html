¿Qué es el Modelo OSI?

El Modelo OSI (Open Systems Interconnection) es un marco de referencia que divide las funciones de una red en 7 capas. Cada capa cumple un rol específico y colabora con las demás para permitir la comunicación entre sistemas de manera estandarizada.

Las 7 Capas del Modelo OSI

Capa 7: Aplicación

Interfaz entre usuario y red. Protocolos: HTTP, FTP, SMTP, DNS.

Capa 6: Presentación

Traducción, encriptación y compresión de datos. Ejemplos: SSL/TLS, JPEG, MP3.

Capa 5: Sesión

Establece, mantiene y finaliza sesiones entre aplicaciones.

Capa 4: Transporte

Entrega confiable de datos de extremo a extremo. Protocolos: TCP, UDP.

Capa 3: Red

Direccionamiento y enrutamiento de paquetes. Protocolos: IP, ICMP, RIP.

Capa 2: Enlace de Datos

Control de acceso al medio y detección de errores. Tecnologías: Ethernet, Wi-Fi.

Capa 1: Física

Transmisión de bits en el medio físico. Ejemplos: cables, señales eléctricas, hardware.
