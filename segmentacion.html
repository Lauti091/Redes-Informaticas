Segmentación de Redes

Organización eficiente y subnetting de redes de datos

¿Qué es la Segmentación de Redes?

Para que las redes funcionen de forma eficiente, sean estables y se puedan aprovechar las tecnologías utilizadas, estas se diseñan y organizan en segmentos pequeños que se comunican con otros segmentos.

🚀

Eficiencia

Reduce congestiones y mejora el rendimiento general de la red

🔒

Seguridad

Mejora el manejo de la seguridad mediante aislamiento de segmentos

🔧

Mantenimiento

Facilita el aislamiento y resolución de problemas

📊

Organización

Establece caminos claros y dirige el tráfico eficientemente

Backbone: Los caminos que transportan el tráfico entre segmentos se denominan 'backbones' o segmentos de red 'troncales'.

Subnetting - Subdivisión de Redes

Cuando se diseña una red se debe tener en cuenta muchos factores como las áreas de la organización, la cantidad de hosts y las subredes necesarias.

Beneficios del diseño previo:

  • Escalar la red (aumentar número de hosts y subredes)
  • Reducir el tamaño de los dominios de broadcast
  • Hacer la red más manejable administrativamente
  • Controlar el tráfico mediante listas de control de acceso (ACL)

Creando Subredes - "Robando Bits"

Red original: 192.168.0.0/24

Máscara /24: 11111111.11111111.11111111.00000000 = 255.255.255.0
Máscara /28: 11111111.11111111.11111111.11110000 = 255.255.255.240

Resultado con /28:

Bits para red: 28 bits
Bits para hosts: 4 bits
Hosts por subred: 2⁴ - 2 = 14 hosts
Número de subredes: 16 subredes

Ejemplos de subredes:

Subred 192.168.0.0/28

ID de red: 192.168.0.0

Primer host: 192.168.0.1

Último host: 192.168.0.14

Broadcast: 192.168.0.15

Subred 192.168.0.16/28

ID de red: 192.168.0.16

Primer host: 192.168.0.17

Último host: 192.168.0.30

Broadcast: 192.168.0.31

CIDR - Redes sin Clases

Con la implementación del diseño bajo CIDR (Classless Inter-Domain Routing), no existen clases fijas. Una red se define por la cantidad de bits que se le otorguen:

190.0.0.0/26 = Red con máscara de 26 bits
10.1.1.0/26 = Otra red con máscara de 26 bits
